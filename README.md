# Code Editor (React) - Project

## Features

1. Compilation and Execution of multiple programming languages
2. Swithable themes for the Code Editor Window

## Working

1. npm install
2. Generate API Keys for Judge0 (Rapid API) and store in a .env file
3. npm start
